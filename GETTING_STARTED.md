# Getting Started with Manuscript

Welcome to Manuscript - a distraction-free writing app with AI-powered assistance.

## Quick Start

1. **Launch Manuscript** - You'll see a blank page with a blinking cursor
2. **Create your first sheet** - Click the pen icon in the toolbar
3. **Start writing** - Just type! Everything is automatically saved

## Navigation

### Sidebar
- **Toggle sidebar** - Use View menu → Show/Hide Sidebar
- **Create sheets** - Click "+" button at bottom of sidebar
- **Organize** - Create groups to organize your sheets
- **Favorites** - Right-click any sheet → Add to Favorites

### Editor
- **Format tools** - Click "Aa" button to show formatting toolbar
- **AI features** - Click brain icon to show AI toolbar
- **Preview** - Click eye icon to toggle Markdown preview
- **Focus mode** - Click eye icon in main toolbar to hide distractions

## Markdown Support

Manuscript supports full Markdown syntax:

### Text Formatting
- `**bold text**` - **Bold**
- `*italic text*` - *Italic*
- `` `code` `` - Inline code

### Headings
- `# Heading 1` - Large heading
- `## Heading 2` - Medium heading
- `### Heading 3` - Small heading

### Lists
- `- item` or `* item` - Bullet list
- `1. item` - Numbered list

### Other
- `> quote` - Blockquote
- `` ```code``` `` - Code block
- `[link text](url)` - Links
- `$LaTeX$` - Inline math (requires web preview)
- `$$LaTeX$$` - Display math (requires web preview)

## AI Features (Requires macOS 26.0+ with Apple Intelligence)

### Enabling AI
1. Ensure you have macOS 26.0+ with Apple Intelligence enabled
2. Click the brain icon in the toolbar to show AI features
3. AI buttons become active when available

### AI Toolbar Buttons

| Button | Action | Description |
|--------|--------|-------------|
| ✓ | Proofread | Check grammar, spelling, clarity |
| ✎ | Rewrite | Rewrite text in different styles |
| + | Summarize | Create concise summary |
| ⭐ | Generate | Generate new content from prompt |
| → | Continue | AI continues your writing |
| ⊕ | Expand | Expand text with more detail |
| 🔊 | Change Tone | Change writing tone |
| ☰ | Outline | Generate document outline |
| 👤 | Character | Develop character profiles |
| 📈 | Plot | Analyze plot structure |

### Using AI Features
1. **Select text** (or place cursor in empty area for document-wide AI)
2. **Click AI button** or use keyboard shortcut
3. **Configure options** (style, tone, prompt) in the panel
4. **Apply result** - Replace selection or insert at cursor

### AI Analysis
- **Sidebar AI button** - Click "Analyze with AI" for tension, pacing, hooks
- **AI Stats** - View readability, sentiment, word count
- **AI Chat** - Ask questions about your writing
- **AI Suggestions** - Get writing improvement tips

## Export

### PDF Export
1. Click the export button (↑) in the editor toolbar
2. Choose save location
3. PDF includes formatting, math, and images

## Settings

Access via **Manuscript → Settings** or `Cmd+,`:

### General Tab
- **Launch at Login** - Auto-start Manuscript
- **Auto-save** - Automatically save changes
- **Appearance** - System/Light/Dark mode

### Editor Tab
- **Font Size** - Adjust text size (12-24pt)
- **Line Spacing** - Adjust spacing (4-12pt)
- **Line Numbers** - Show/hide line numbers

### AI Tab
- **Status** - Check if Apple Intelligence is available
- **Features** - Overview of all AI capabilities

## Tips

- **Minimal by default** - Toolbars collapse to keep you focused
- **Keyboard shortcuts** - Use `Cmd+Shift+Return` for quick proofread
- **Auto-title** - Sheet titles auto-update from first line
- **Markdown preview** - Toggle with eye icon to see formatted output
- **Focus mode** - Hides sidebar and toolbar for true full-screen writing

## Troubleshooting

### AI Features Not Available
- Ensure macOS 26.0+ is installed
- Enable Apple Intelligence in System Settings
- Check AI status in Settings → AI tab

### PDF Export Issues
- Ensure you have write permissions to save location
- Try web preview mode for complex documents with math

### Sidebar Missing
- Use View menu → Show Sidebar
- Click sidebar toggle in toolbar (if visible)
